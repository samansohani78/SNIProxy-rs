About HTTP/1.1

HTTP/1.1 is the first standardized version of HTTP that was widely adopted. It introduced several performance improvements over HTTP/1.0:

  • Persistent connections (keep-alive)
  • Chunked transfer encoding
  • Request pipelining
  • Additional caching mechanisms
